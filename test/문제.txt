[1] 메인화면
  1. 메인화면 전체 레이아웃은 css grid 기술을 적용하시오.
  2. 메인화면 부분 레이아웃은 flex 기술을 적용하시오.
  3. 시멘틱 태그사용(header, nav, main, section, aside, footer)
  4. 최상위메뉴 - 가로 100%, 세로 50px
   > 메뉴표시(로그인 | 회원가입)
   > 글자크기 : 0.8rem, 글자색 : 흰색, 글자두께 : 굵게
  5. 헤더 - 가로 : 600px, 세로 : 150px
   > 배경이미지는 픽사베이에서 사진을 다운받아 적용한다.
  6. 메인메뉴 - 가로 : 600px, 높이 : 50px
   > 웹표준, 자바, 안드로이드, 데이터베이스, 스프링
   > UL, LI, A태그 사용하여 구현한다.
   > 메뉴는 밑줄제거, 마우스 오버시 밑줄 적용
   > 글자색 : 흰색, 글자크기 : 0.8rem, 글자굵기 : bold
  7. 메인
   > 좌측 부가정보 - 가로 : 20%적용, 배경색 : lightSkyBlue
   > 우측 컨텐츠 - 가로 : 80%적용, 배경색 : lightcyan
   > 회원가입 양식 - 가로 : 250px적용, 라인간 외부여백 0.3px
  8. 푸터 - 가로 : 600px, 높이 : 70px 적용
   > 아래문구를 가로세로 중앙에 표시
   > 글자크기 : 0.8rem, 글자색 : 흰색, 글자굵기 : 굵게
   > 출력문구 : Cpyright by Gildong Hong. All Rights Reserved.

[2] 회원가입
  1. 회원가입 : h3, 중앙정렬, 자간 : 0.5em, 외부여백 : 10px
  2. 아이디, 별칭: text
  3. 비밀번호&비밀번호확인 : 입력시 '*'로 표시
  4. 전화번호 : tel
  5. 지역 : 콤보박스 (==선택==, 서울, 부산, 대구, 대전, 울산)
  6. 성별 : 택일 (남자, 여자)
  7. 취미 : 다중선택 (독서, 음악감상, 운동, 게임)
  8. 생년월일 : 년월일 선택
  9. 가입버튼 : 배경색-green, 글자색-흰색, 글자굵기-굵게, 내부여백 - 50px
  10. 라벨명 클릭시 input요소 활성화 되도록 구현한다.
